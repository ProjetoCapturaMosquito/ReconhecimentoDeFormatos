# Captura Mosquito

## Instalação

Clone this repository and enter on folder

Create virtual environment and activate

```console
$ pip3 install virtualenv
$ virtualenv venv
$ source venv/bin/activate
```

To deactivate use

```console
$ deactivate
```

Install requirements

```console
pip3 install opencv-python Pillow
```

Run with

```console
$ python3 main.py
```
